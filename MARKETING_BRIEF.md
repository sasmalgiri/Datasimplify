## 1. What Is CryptoReportKit?

CryptoReportKit (CRK) is an all-in-one crypto analytics platform that combines:

1. **71 Live Interactive Dashboards** — real-time crypto dashboards with 75+ chart widgets
2. **85+ Excel Custom Functions** — pull live crypto data directly into Excel/Google Sheets
3. **8 Pro Tools** — tax reports, DCA tracker, custom dashboard builder, P&L tracker, smart screener, multi-exchange portfolio, BTC cycle comparison, 60+ currency support
4. **8 Smart Intelligence Widgets** — algorithmic signals, health scores, risk radar, alpha finder

**Key differentiator:** BYOK (Bring Your Own Key) — users enter their own CoinGecko API key. All data requests go directly from their browser to CoinGecko. We never see, store, or transmit their API keys. This is a massive privacy and trust advantage.
